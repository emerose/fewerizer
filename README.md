# inbox fewerizer

1. install dependencies

install [clasp](https://github.com/google/clasp)

    npm install -g @google/clasp

Enable Google Apps Script API: https://script.google.com/home/usersettings

2. deploy to google

    clasp login

    clasp create --type webapp
    

3. grant permissions

4. test

5. install trigger

6. enjoy
